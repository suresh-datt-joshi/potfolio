# Portfolio Website - Suresh Datt Joshi

## Overview
This is a React-based portfolio website for Suresh Datt Joshi, a Full-Stack Developer & DevOps Enthusiast. The website showcases his skills, projects, education, certificates, and contact information.

## Tech Stack
- **Frontend Framework**: React 18.2.0
- **Build Tool**: Vite 5.0.8
- **Styling**: Tailwind CSS 3.4.0
- **Animations**: Framer Motion 12.23.12
- **Icons**: Lucide React 0.303.0

## Project Structure
```
portfolio-website/
├── public/                    # Static assets
│   ├── profile-picture.jpeg
│   └── suresh_resume.pdf
├── src/
│   ├── assets/               # Images and certificates
│   ├── components/
│   │   ├── common/          # Reusable components
│   │   └── home/            # Section components
│   ├── pages/               # Page components
│   ├── App.jsx              # Main app component
│   └── main.jsx             # Entry point
└── package.json
```

## Development
- **Dev Server**: Running on port 5000
- **Command**: `npm run dev` (from portfolio-website directory)
- **Environment**: Configured for Replit with host 0.0.0.0

## Deployment
- **Type**: Autoscale (stateless website)
- **Build Command**: `npm run build`
- **Preview Command**: `vite preview --host 0.0.0.0 --port 5000`

## Features
- Responsive design with dark theme
- Smooth scrolling navigation
- Sections: Home, About, Skills, Projects, Education, Certificates, Contact
- Downloadable resume
- Social media links
